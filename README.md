# npathways | Global Career Guidance & Portfolio Building

A premium, SEO-optimized career guidance portal built with React and Vite. This platform is designed to architect academic and professional leadership journeys through strategic pathways and expert mentorship.

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/npathways-cmp-pb.git
   ```
2. Navigate to the project directory:
   ```bash
   cd npathways-cmp-pb
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Development

Run the development server:

```bash
npm run dev
```

### Production

Build the application for production:

```bash
npm run build
```
